# AI RAM Guardian — Product Narrative

## One-line pitch

A native macOS glass utility for AI-heavy workstations that measures, explains, and receipts every memory or thermal intervention.

## Buyer-facing pitch

AI coding tools changed the workstation. Devin, Windsurf, Claude, Cursor, VS Code, and Codex run heavy helper processes, language servers, indexers, renderers, and agents. Existing RAM cleaners were built for the pre-agent desktop. They show fake boost numbers, kill the wrong things, and do not prove outcomes.

AI RAM Guardian is different: it treats the Mac like a living AI workstation and writes a receipt for every claim.

## Moat

The moat is not a memory button. The moat is the ledger:

```text
observe → decide → intervene gently → re-measure → write receipt → accumulate proof
```

A competitor can copy a glass panel. Copying a measured intervention ledger requires process sampling, pressure forecasting, thermal sensing, focus-state policy, receipt persistence, and honesty around uncertainty.

## Novel features

### 1. Receipt-native memory care

Every intervention is auditable. The app does not say "boosted." It says what changed, when it changed, and how the measurement was taken.

### 2. FLOW focus-aware resource easing

When the user switches to one AI app, non-focused AI helpers can be gently backgrounded, then restored when focused again. This follows the user's attention rather than blindly trimming memory.

### 3. SOOTHE thermal proof

When thermal state is serious, the app can demote the hottest eligible helper process, then re-read temperature later and write an observational before/after receipt.

### 4. On-device tiny ML

The app does not bundle a heavyweight neural net. It uses the honest statistical learning suitable for a guardian:

- Online Kalman forecast for memory-pressure trajectory and uncertainty.
- Per-app anomaly baselines learned from the local machine.

### 5. Glass UI with self-audit

The UI shows the app's own RAM and CPU cost. Efficiency is not a marketing claim; it is visible.
